# For2048
Clone of 2048 game in Swift
# Discription
A Funny Time Killing Game. Swipe to move the cubes, try to earn the largest-digit cube as you can.
# IDE
Xcode 12.1

Swift 5.3

Better experience with iPhone 8 or later devices.
